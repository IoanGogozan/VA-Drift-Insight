# Demo Script In Norwegian

Jeg har laget en enkel demo for å vise hvordan praktisk VA-drift kan kobles med data, kart og beslutningsstøtte.

Målet er ikke å erstatte fagfolk eller eksisterende driftskontrollsystemer, men å vise hvordan data fra ledningsnett, pumpestasjoner, målere, hendelser og nedbør kan brukes til bedre prioritering.

Her ser vi først en risikooversikt for en fiktiv kommune. Systemet viser lekkasjerisiko, mistanke om fremmedvann, datakvalitet og anbefalte feltkontroller.

I lekkasjekartet kan vi klikke på en målesone. Forklaringen viser at området har økt nattforbruk, gamle støpejernsledninger og tidligere lekkasjer i nærheten. Systemet foreslår derfor akustisk lekkasjesøk og ventilkontroll.

På avløpssiden ser vi pumpestasjon PS-03. Pumpetiden øker tydelig etter nedbør og holder seg forhøyet etter at regnet stopper. Dette kan tyde på innlekking eller feilkoblet overvann oppstrøms.

Til slutt genererer systemet en rapport som kan brukes i møte, planlegging eller som grunnlag for videre undersøkelser.

Min bakgrunn er en kombinasjon av praktisk VVS/VA-forståelse og backend-utvikling. Derfor kan jeg bidra til å gjøre eksisterende VA-data mer nyttige for drift, lekkasjekontroll og prioritering.

## Short Pitch

Jeg kan være personen som kobler felt, drift og data.
